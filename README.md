# Paarth Sharma — Portfolio

Professional portfolio website built with React, TypeScript, Vite, and Tailwind CSS.

## Live Site
[paarth-sharma.com](https://paarth-sharma.com)

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Cloudflare Pages

## Features
- Responsive single-page portfolio
- Mobile-friendly navigation
- Projects, experience, skills, and contact sections
- Resume download
- Fast static deployment via Cloudflare Pages

## Local Development
```bash
npm install
npm run dev
```
